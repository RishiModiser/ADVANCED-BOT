## ğŸ”§ Troubleshooting

### Application Won't Start

**Windows:**
- Right-click the file â†’ "Run as Administrator"
- Check if antivirus is blocking it (add to exceptions)
- Make sure you have the .exe file, not a shortcut

**Linux/Mac:**
1. Open Terminal
2. Run: `chmod +x ADVANCED-BOT`
3. Then: `./ADVANCED-BOT`

### Browser Download Fails

- **Check internet connection** is active
- **Check firewall** isn't blocking downloads
- **Try running again** - it will retry automatically
- **Manual install**: If automatic download fails repeatedly:
  ```bash
  # Windows (in Command Prompt)
  cd %USERPROFILE%\.cache\ms-playwright
  
  # Linux/Mac (in Terminal)
  cd ~/.cache/ms-playwright
  ```
  Then contact support with error details

### Application Crashes

1. **Check system resources**:
   - Close other applications
   - Ensure at least 4GB RAM available
   - Check disk space (need 500MB+)

2. **Run from terminal/command prompt** to see error messages:
   - **Windows**: Open cmd, run `ADVANCED-BOT.exe`
   - **Linux/Mac**: Open terminal, run `./ADVANCED-BOT`

3. **Report the issue**:
   - Visit: https://github.com/RishiModiser/ADVANCED-BOT/issues
   - Include: OS version, error messages, steps to reproduce

### Slow Performance

- **Reduce concurrent threads** (fewer browsers at once)
- **Check RAM usage** (Task Manager / Activity Monitor)
- **Disable advanced interactions** if not needed
- **Close other applications**
