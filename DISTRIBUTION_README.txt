## üîß Troubleshooting

### Application Won't Start
- **Windows**: Right-click and "Run as Administrator"
- **Linux/Mac**: Check execute permissions: `chmod +x ADVANCED-BOT`
- Check antivirus isn't blocking the application

### Browser Download Fails
- Ensure internet connection is active
- Check firewall settings
- Try running the application again (it will retry)

### Application Crashes
- Check system resources (RAM/CPU)
- Try closing other applications
- Run from terminal to see error messages:
  - Windows: `ADVANCED-BOT.exe` in cmd
  - Linux/Mac: `./ADVANCED-BOT` in terminal
